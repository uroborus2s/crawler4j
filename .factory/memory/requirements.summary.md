# Requirements Summary

- `REQ-001`: Root desktop app must have a real, release-aligned entrypoint
- `REQ-002`: Core must be able to execute module workflows without legacy missing imports
- `REQ-003`: SDK, Contracts, and CLI must remain buildable and usable
- `REQ-004`: Release metadata and docs must align to a single version truth
- `REQ-005`: Project must operate inside the software-factory lifecycle with tracked workitems
