## 配色方案

采用**深色主题**，减少长时间使用的视觉疲劳：

| 元素 | 颜色 |
|------|------|
| 背景 | #1e1e2e |
| 侧边栏 | #181825 |
| 卡片背景 | #313244 |
| 主文字 | #cdd6f4 |
| 次要文字 | #a6adc8 |
| 主色调 | #89b4fa (蓝) |
| 成功 | #a6e3a1 (绿) |
| 警告 | #f9e2af (黄) |
| 错误 | #f38ba8 (红) |
| 禁用 | #6c7086 |

## 交互规范

1. **按钮反馈**: 所有按钮点击后需有 loading 状态或即时反馈
2. **表格分页**: 默认每页 20 条，支持 20/50/100 切换
3. **搜索防抖**: 搜索输入 300ms 防抖
4. **确认操作**: 删除、置黑等危险操作需二次确认
5. **Toast 提示**: 操作成功/失败显示右上角 Toast，3 秒自动消失
6. **表单验证**: 实时校验，错误时输入框标红并显示错误信息
