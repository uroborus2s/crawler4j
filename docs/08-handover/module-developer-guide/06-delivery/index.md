# 06 交付与验收

很多新手会把“DevLink 下已经调通”误当成“模块已经可以交付”。  
这是不够的。

`DevLink` 只能证明你的源码目录在开发模式下能跑，不能证明正式安装链路没有问题。  
真正的交付闭环至少还要补上：

1. 打出符合结构要求的 zip
2. 在宿主应用里完成安装
3. 跑一次安装后的 smoke

## 阅读目标

读完本章后，你应该知道：

1. 为什么正式交付物不是 wheel 而是 zip。
2. zip 结构错在哪里最容易出问题。
3. 安装后为什么 `🐞 调试` 可能会消失。
4. 什么样的检查才算“我已经可以把这个模块交给别人了”。
