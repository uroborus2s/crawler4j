# 07 排错

这一章是“卡住时先看这里”的集中答疑。

如果你已经按前面的步骤做了，但还是遇到：

- 命令不工作
- 模块不被识别
- 调试按钮不出现
- zip 安装失败
- UI 不显示

先不要急着改代码，先把这一章过一遍。  
很多问题并不是代码逻辑错误，而是运行链路上的某个条件没有满足。

## 阅读目标

读完本章后，你应该能更快判断问题属于哪一类：

1. CLI 使用问题
2. 模块结构问题
3. 策略配置问题
4. 调试链路问题
5. 安装结构问题
6. UI trust gate 问题
