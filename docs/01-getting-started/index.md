# 入门说明概览

本目录面向第一次接手 `crawler4j` 的人，优先回答三个问题：这是什么项目、文档应该怎么读、我下一步去哪一层。

## 本目录包含什么

- `project-overview.md`：项目定位、仓库组成和当前运行事实。
- `quick-start.md`：最小启动步骤和继续阅读路径。
- `document-map.md`：四大模块边界和按角色阅读顺序。

## 推荐阅读顺序

1. 先读 [文档地图](document-map.md)，确定自己属于哪一类读者。
2. 再读 [项目概览](project-overview.md)，理解系统边界和当前组成。
3. 最后按 [快速开始](quick-start.md) 或下游角色入口继续阅读。

## 维护边界

- 根 `docs/index.md` 负责声明页面路径和权限。
- 本目录只负责解释“从哪里开始看”，不重复维护全站导航。
